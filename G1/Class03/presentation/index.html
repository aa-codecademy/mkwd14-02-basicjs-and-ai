Switch statements

  • Starts with a variable called switch value
  • Each case indicates a possible value for that variable
  • Have a default option if no match is found

Structure


switch(someVal) {
    case 'value1':
        //do something;
        break;
    case 'value2':
        //do something else;
        break;
    default:
        //no value match found, do default option
        break; //this is optional
}

"fall through" scenario

  • If you omit break from a case, and that case matches or runs, execution will continue with the next case's statements regardless of that case matching


switch(someVal) {
    case 'value1':
    case 'value2':
        //do something;
        break;
    case 'value3':
        //do something else;
        break;
    default:
        //no value match found, do default option
        break;
}

What is a function?

  • Break up the code's tasks into reusable pieces
  • Generally a named section of code that can be "called" by name, and the code inside it will be run each time

Function declaration

  • Put the keyword function, give it a name and then write the statements needed to achieve its task inside curly braces (yes, a code block indeed)

function(){
    //... some code that does something
}

Function execution

  • Or calling the function
  • Execute all statements between function's curly braces with just one line of code
  • ...as many time as you want...


function myFunction() {
    //loops, ifs, statements, anything;
    //...
}

myFunction();
myFunction();
myFunction();

  • Code in function won't run, unless explicitelly called
  • Doesn't matter where are defined
    • Best practice: define your functions before you call them

Functions with arguments

  • Pieces of data to work with in your function
  • In the parenthesis:
    • Define how many pieces of data
    • How am I going to call them
  • Do not need the words let/const


//creating function with arguments

function myFunction(x, y) {
    let myVar = x * y;
    console.log(myVar);
}

myFunction(1,3);
myFunction(2,5);
myFunction(100,232131);

Returning values


function myFunction(x, y) {
    let myVar = x * y;
    console.log(myVar);
    return myVar;
}

myFunction(1,3);
myFunction(2,5);

Scope

  • Scope determines the accessibility (visibility) of variables.
  • Variables defined inside a function are not accessible from outside.
  • A collection of variables and rules for accessing them.
  • JavaScript has function scope: Each function creates a new scope.
  • No matter where or how a function is invoked, its scope is defined by where it was declared.

Function Scope

Variable scope


function simpleFunction() {
    let foo = 500;  // local variable
    console.log(foo); // 500
}

simpleFunction();
console.log(foo);  // undefined

Variable scope


let foo = 500; // global variable
function simpleFunction() {
    let foo = 600; // local variable
    console.log(foo); // 600
}

simpleFunction();
console.log(foo); // 500


Variable scope

  • When you declare a variable, it is available anywhere in that scope and inner scopes.


let foo = 500; // global variable
function simpleFunction() {
    foo = 600; // modifies global variable
    console.log(foo); // 600
}

simpleFunction();
console.log(foo); // 600

Variable scope

  • If you access a variable outside its scope, you get ReferenceError
  • Assigning to undeclared variable creates a global variable (bad practice)

Variable scope


function foo() {
    a = 1; // not declared
}

foo();
console.log(a); // 1 -- auto global

  • Description: Celsius to Fahrenheit formula: X簞C x 1.8 + 32

  • Bonus: Write another function to convert Fahrenheit to Celsius: (5/9) * (F-32)

Exercise 2 Solution


// Declaration of both functions
function celToFar(celsius){
    return celsius * 1.8 + 32;
}

function farToCel(fahrenheit){
    return (5/9) * (fahrenheit-32);
}

// Prompt user
let personChoice = prompt("Enter F for Fahrenheit or C for Celsius");
let personInput = parseInt(prompt("Please enter value:"));

if(personChoice === "F"){
    console.log(`${farToCel(personInput)}C`);
} else if(personChoice === "C"){
    console.log(`${celToFar(personInput)}F`);
} else {
    console.log("ERROR!");
}
