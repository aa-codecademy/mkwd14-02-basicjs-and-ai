### 1. Average of Numbers

**Goal:**
Ask the user for how many numbers they want to enter, store them in an array, and calculate the average.

**Function:**
`function calculateAverage()`

**Input:**

* Number count
* Numbers

**Output:**
Average value

**Test Cases:**

* Input: `3, 10, 20, 30` → Output: `20`
* Input: `4, 5, 5, 5, 5` → Output: `5`
