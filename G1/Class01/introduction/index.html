What is JavaScript?

  • Developed by Brendan Eich in 1995, first known as LiveScript
  • A lightweight programming language that runs in a Web browser
  • Interpreted, not compiled
  • JavaScript is NOT Java

What is JavaScript?

Javascript History

What Can JavaScript Do?

  • With JavaScript web pages are dynamic and interactive
  • JavaScript can put dynamic text into an HTML page
  • JavaScript can react to events
  • JavaScript can be used to validate input data
  • Allows parts of the page to appear, disappear, or move dynamically

How It works?

Writing a JavaScript Program?

  • The Web browser runs a JavaScript program when the page is first loaded or in response to events.
  • Programs can be placed in HTML or external files
  • Placing in external files allows hiding code from the user
  • Code in HTML can be viewed by anyone

VARIABLES

  • Variables are used to store data in memory
  • Two steps required: Declare and Initialize

let number; // Declaration
number = 5; // Initialization
let number = 5; // Declaration and Initialization

Variables in Memory

VARIABLES

  • Variable names must begin with a letter or underscore
  • Variable names are case-sensitive
  • Variables are untyped (VALUE IS THE TYPE!)
  • Local vs Global variables explained

Primitive Data Types

  • Strings, Numbers, Undefined, Boolean, NULL

Undefined vs Null

let testVariable; alert(testVariable); // undefined alert(typeof testVariable); // undefined let testVariable = null; alert(testVariable); // null alert(typeof testVariable); // object null === undefined; // false null == undefined; // true null === null; // true
